# 手动启动前后端

## 1. 启动前检查

先确认旧进程已经停掉，否则你看到的还是旧代码和旧缓存。

- 后端：结束正在跑的 `python` / `uvicorn` 进程
- 前端：结束正在跑的 `node` / `vite` 进程
- 如果开了 Redis，保留也行，但遇到脏响应时建议先清一次缓存

## 2. 启动后端

在项目根目录下执行：

```powershell
cd a-stock-info\backend
python -m uvicorn main:app --host 127.0.0.1 --port 8001 --reload
```

如果你的环境里 `python` 不是可用命令，就用你本机对应的 Python 启动器。

## 3. 启动前端

另开一个 PowerShell 窗口：

```powershell
cd a-stock-info\frontend
npm run dev -- --host 127.0.0.1 --port 5173
```

## 4. 验证

先看后端健康检查：

```powershell
Invoke-RestMethod http://127.0.0.1:8001/health
```

再查 K 线接口：

```powershell
Invoke-RestMethod "http://127.0.0.1:8001/api/v1/stocks/601991.SH/kline?period=daily"
```

正常情况下，日 K 不应该只返回 1 条。

## 5. 如果你仍然看到 1 条 K 线

按这个顺序处理：

1. 先停掉旧后端和旧前端进程
2. 重新启动后端
3. 重新启动前端
4. 浏览器里强制刷新一次

如果还不对，说明跑的不是这份代码，或者后端进程没真正重启成功。
